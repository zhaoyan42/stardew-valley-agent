星露谷物语 AI 详细设计 (v2)

针对你的反馈，我们引入了成熟的记忆方案，并明确了层级职责。

1. 成熟记忆方案 (Mem0 + ChromaDB)

不再手动写 JSON 文件，而是采用成熟的开源方案：

  • Mem0 (Memory Layer): 自动从对话和事件中提取事实（如：“用户喜欢钓鱼”，“今天死在了10层”）。
  • ChromaDB (Local DB): 作为本地持久化存储，支持向量检索，让 AI 能根据当前情境“想起”相关经验。
  • 自我演进：LLM 会定期 review 记忆库，自动更新战术权重。

2. 战术层寻路 (A* Pathfinding)

空间逻辑完全下放到 C# Mod 层：

  • 语义指令：LLM 只发 `{"action": "go_to", "target": "Blacksmith"}`。
  • 本地执行：Mod 调用 A* 算法，避开杂草、NPC 和栅栏。
  • 实时避障：战术层处理移动过程中的微调。

3. 动态战术规则 (Tactical JSON)

LLM 下发的 `tactics.json` 示例：

{
  "combat": { "kiting_distance": 1.5, "heal_at_hp": 30 },
  "inventory": { "auto_trash_low_value": true },
  "social": { "auto_gift_on_meet": true }
}

4. 长期规划初始化 (Active Inquiry)

系统首次启动时，AI 将主动询问并记录：

  • 终极目标：全成就、全图鉴、还是纯粹赚钱？
  • 社交偏好：优先攻略哪个 NPC？
  • 经营风格：极致工业化还是自然农耕？
